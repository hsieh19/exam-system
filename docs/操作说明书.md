# 企业考试系统操作说明书

## 1. 系统简介

企业考试系统是一个简洁、高效的企业内部在线考试平台。它支持多种题型、智能组卷、自动评分、多维度成绩分析以及灵活的权限管理。本系统采用现代化的 Web 技术构建，界面美观，操作便捷。

---

## 2. 登录系统

### 2.1 访问地址
打开浏览器（推荐使用 Chrome 或 Edge），访问系统地址：`http://localhost:3000`

### 2.2 账号登录
在登录页面输入您的用户名和密码。

**默认账号示例：**

| 角色 | 用户名 | 密码 | 权限说明 |
| :--- | :--- | :--- | :--- |
| **管理员** | `admin` | `admin123` | 拥有系统的所有权限，包括用户管理、题库维护、试卷发布等。 |
| **考生** | `张三` | `123456` | 仅能参加被分配的考试，查看自己的历史成绩和排名。 |

### 2.3 主题切换 🎨
系统支持多种个性化主题，点击右上角的 **主题图标** 即可切换：
*   🌙 **深色**：默认专业深色模式，适合长时间工作。
*   ☀️ **浅色**：明亮清爽，高对比度。
*   🌊 **深蓝**：冷静沉稳的商务风格。

---

## 3. 考生操作指南

### 3.1 参加考试
1.  登录后进入 **"考试"** 页面。
2.  在考试列表中找到状态为 **"进行中"** 的考试。
3.  点击卡片右侧的 **"开始考试"** 按钮进入答题界面。
4.  **答题界面说明**：
    *   **顶部信息栏**：显示当前题号、考试进度、本题剩余时间及累计用时。
    *   **题目显示区**：显示题目类型（单选/多选/判断）、题干及选项。
    *   **操作区**：选择答案后，点击 **"下一题"**。
5.  完成所有题目后，系统会自动提交试卷并计算分数。
    *   *注意：如果单题倒计时结束，系统会自动跳转到下一题。*

### 3.2 查看成绩与排名
1.  切换到 **"得分及排行榜"** 页面。
2.  在下拉框中选择您参加过的试卷。
3.  系统将显示您的考试成绩、排名以及前三名的光荣榜。

## 4. 管理员操作指南

管理员登录后，侧边栏将显示完整的功能菜单。

### 4.1 分组及用户设置
在此页面管理组织架构和人员。
*   **分组管理**：点击左侧 **"添加分组"** 创建部门（如"研发部"、"市场部"）。
*   **用户管理**：
    *   点击 **"添加用户"**，填写姓名、账号、密码、角色（管理员/普通用户）及所属分组。
    *   可以通过搜索框快速查找用户。
    *   支持重置用户密码或删除离职人员账号。

### 4.2 题库设置
管理系统的试题资源。
*   **筛选与检索**：使用顶部的筛选器，按专业、题型或设备类型查找题目。
*   **添加题目**：
    *   点击 **"增加单选题/多选题/判断题"**。
    *   填写题干、选项（支持Markdown格式）、正确答案以及题目解析。
    *   设置题目所属的专业分类。
*   **导入/导出**：支持 Excel 模板批量导入题目，也支持将现有题库导出备份。

### 4.3 创建试卷
发布新的考试任务。
1.  **基本信息**：输入试卷名称。
2.  **规则设置**：
    *   添加规则行，指定从某个专业中抽取多少道某种类型的题目。
    *   设置每题分值、漏选得分规则（针对多选）及每题答题时限。
    *   点击 **"校验试卷规则"** 确保题库中有足够的题目。
3.  **生成试卷**：
    *   **自动生成**：系统随机抽题组卷。
    *   **手动选择**：您亲自从题库中挑选题目。
4.  保存后，试卷即可在考试列表中看到。

### 4.4 考试分析
查看考试的整体情况和详细数据。
*   选择试卷后，可查看平均分、最高分、最低分等统计指标。
*   查看所有参考人员的详细答题记录和得分明细。
*   支持清空某次考试的记录（慎用）。

### 4.5 系统日志
审计系统的关键操作。
*   记录了登录、增删改查等所有敏感操作。
*   支持按操作对象（用户/试卷/题目）、操作类型、日期范围进行筛选。
*   提供日志清理功能。

### 4.6 数据库设置
高级系统维护功能。
*   **SQLite**（默认）：轻量级，数据存储在本地文件。支持导入/导出 `.db` 文件进行备份。
*   **MySQL / PostgreSQL**：支持连接外部企业级数据库，适合大规模部署。

## 5. 常见问题 (FAQ)

**Q: 忘记密码怎么办？**
A: 请联系管理员，在"分组及用户设置"页面为您重置密码。

**Q: 考试中途网络断开会怎样？**
A: 系统会自动保存您的答题进度。重新登录后，可以继续之前的考试，但计时器可能不会停止，请尽快恢复连接。

**Q: 如何修改系统的主题颜色？**
A: 点击页面右上角的主题图标，选择您喜欢的风格即可。所选主题会自动保存，下次登录时依然生效。
